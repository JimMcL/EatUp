This directory contains a game-like web application which requires
participants to view a series of photos of arthropods, and decide, for
each one, whether or not it is an ant.

It is (temporarily) in this location because it is served up by the
sampleIt web server, which means it can use cookies and be accessed
from other machines.

URL: http://<ip>/test/index.html
http://127.0.0.1/test/index.html
http://192.168.0.15/test/index.html
